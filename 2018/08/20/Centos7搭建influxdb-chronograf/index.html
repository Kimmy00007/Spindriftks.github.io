简单语法

influxdb与传统数据库的比较

influxdb MySQL
database 数据库
measurement 数据库中的表
points 表里面的一行数据

influxdb数据的构成：
Point由时间戳（time）、数据（field）、标签（tags）组成。

Point属性 传统数据库中的概念
time 每个数据记录时间，是数据库中的主索引(会自动生成)
fields 各种记录值（没有索引的属性）也就是记录的值：温度， 湿度
tags 各种有索引的属性：地区，海拔
