zabbix触发器函数说明

触发器函数官方文档

函数

abschange

描述：最近获取值与之前获取值差的绝对值。

参数：

说明：

avg (sec|#num,)

描述：指定评估期内一个项目的平均值

参数：

说明：

band (sec|#num,mask,)

描述：项目值和掩码的按位与值。

说明：

参数：

change

描述：最近获取值与之前获取值的差。

说明：

参数：

count (sec|#num,,,)

描述：指定评估期内值的个数。

说明：

参数：

date

描述： 当前日期，以YYYYMMDD格式表示。

说明：

参数：

dayofmonth

描述：当前是本月的第几天，取值范围从1到31。

说明：

参数：

dayofweek

描述：当前是本周的第几天，取值范围从1到7 (周一 - 1, 周日 - 7)。

说明：

参数：

delta (sec|#num,)

描述：指定评估期内最大值和最小值的差(‘max()’ 减去 ‘min()’)。

说明：

参数：

diff

描述：比较最近获取值与之前获取值是否相同。

说明：

参数：

forecast (sec|#num,,time,,)

描述：项目的未来值，最大值，最小值，差值或平均值。

说明：

参数：

fuzzytime (sec)

描述：检查项目时间戳和zabbix服务器时间相差多大。

说明：sec - 秒数

参数：

iregexp (pattern,)

描述：该函数和 regexp() 类似，只是不区分大小写。

说明：see regexp()

参数：支持值的类型: str, log, text

last (sec|#num,)

描述：最近的值。

说明：

参数：

logeventid (pattern)

描述：检查最近日志记录的EventID是否匹配正则表达式。

说明：pattern - 使用正则表达式表示需要匹配的模式， POSIX extended 类型。

参数：

logseverity

描述：最近日志记录的日志等级。

说明：

参数：

logsource (pattern)

描述：检查最近的日志记录是否匹配参数的日志来源。

说明：pattern - string类型

参数：

max (sec|#num,)

描述： 指定评估期内一个项目的最大值。

说明：

参数：

min (sec|#num,)

描述：指定评估期内一个项目的最小值。

说明：

参数：

nodata (sec)

描述：检查是否没有接收到数据。

说明：

参数：

now

描述：距离Epoch (00:00:00 UTC, January 1, 1970)时间的秒数。

说明：

参数：支持值的类型: any

percentile (sec|#num,,percentage)

描述：P-th 一段时间的百分值， P (percentage) 做为第三个参数。

说明：

参数：

prev

描述：前一个值。

说明：

参数：

regexp (pattern,)

描述：检查最近的值是否匹配正则表达式。

说明：

参数：

str (pattern,)

描述：从最新值中查找一个字符串。

说明：

参数：

strlen (sec|#num,)

描述：最新值的字符长度（而不是字节数）。

说明：

参数：

sum (sec|#num,)

描述：指定评估期内项目值的和。

说明：

参数：

time

描述：当前时间，以HHMMSS格式表示。

说明：

参数：

timeleft (sec|#num,,threshold,)

描述：达到阀值需要多久时间。

说明：

参数：
